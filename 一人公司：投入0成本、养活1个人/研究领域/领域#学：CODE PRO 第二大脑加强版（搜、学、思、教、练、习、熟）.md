## 费曼学习法（Feynman Technique）

https://baike.baidu.com/item/%E8%B4%B9%E6%9B%BC%E5%AD%A6%E4%B9%A0%E6%B3%95/50895393

是==一种高效学习方法，由诺贝尔物理学奖得主理查德·费曼提出，核心思想是**“输出倒逼输入”**，即通过**用简单、清晰的语言向别人（或假想的听众）讲解一个知识点，来发现自己理解上的不足，并不断精简和完善，从而真正掌握知识**==。它包含四个主要步骤：选定目标、教学讲解、回顾反思、简化优化。

费曼学习法可以简化为四个单词：Concept （概念）、[Review](https://baike.baidu.com/item/Review/10921488?fromModule=lemma_inlink) （回顾）、Simplify （简化）、Teach （教给别人）。

### 核心步骤

1. **选择一个概念（Concept）**：确定要学习的知识点或概念。
2. **尝试教学（Teach）**：假设自己是老师，用最简单的语言（甚至对小孩能听懂）向别人复述这个概念。输出是检验理解的关键。
3. **回顾和发现（Review）**：在复述过程中，留意自己卡住、含糊不清或需要查阅的地方，这是知识的薄弱环节。
4. **简化和优化（Simplify & Refine）**：回到原始材料，填补知识漏洞，找到更精炼的表达方式，将复杂的概念变得通俗易懂，直到能流畅地讲给外行听。


## 苏格拉底教学法（Socratic Method）

又称“产婆术”或“问答法”，是==一种通过教师不断提问引导学生思考、发现自身认知中的矛盾，从而主动构建知识、得出真理的教学方法==。 核心在于不直接给予答案，而是扮演“思想助产士”角色，激发学生独立思考，培养其探究能力、批判性思维和获取智慧的过程。

### 核心原则

- **质疑与探究**： 
    
    通过一系列有系统、有逻辑的提问，引导学生深入分析概念、揭示假设、探究想法的本质。 
    
- **启发与引导**： 
    
    不填鸭式灌输，而是让学生在问答互动中，自己发现问题、修正错误、逐步接近真理。 
    
- **师生平等**： 
    
    营造轻松的讨论氛围，教师与学生共同参与，互相启发，教师亦能从学生那里学习。 
    

- **“精神助产术”**： 
    
    苏格拉底自比助产士，帮助学生“生出”自己的思想，而非强加知识
