# 【Notion】网络资源索引

- ！！！致命缺点：没有Tag！只能用页面引用和反向链接简单模拟一下
- ！缺点：搜索功能较弱
- ！缺点：浏览器版右键没有翻译！
- 缺点：页面内搜索有时搜不到
- 缺点：无法实现浏览次数自动增加、搜索按浏览次数排序
- 左侧边栏能显示页面列表和拖到调整顺序和层级。另外，由于每个页面和节点都有文本网址，所以可以方便地配合浏览器书签栏，实现更自由的**渐进式树状结构**
- 缺点：左侧边栏无法区分是否含有子页面！也无法全部展开子页面！需逐个打开才能知道
- 优点：可以发布某个网页

# 【O**bsidian**】**顶级知识管理神器！双链笔记先驱**！

双向链接、插件、本地AI、纯文本（方便迁移）、关系图谱、思维导图、抽认卡数据库

Dataview是table标签（可系统配置table来美化外观）：JS（实现任何功能！！！）、Query

base是div标签（不可系统配置table来美化外观！只能CSS代码）

不适合：信息搜集（如印象笔记）、Word等排版文档、时间管理

适合：码字、深度知识网络（知识节点、双向链接）

云同步（收费）！但可使用免费的如NAS、Github、OneDrive、Google Drive等云存储实现云同步

可发布（收费）

优点：所有文件存放在本地（如NAS），不受空间大小限制、且安全，访问速度超快

缺点：只能本地大模型或AI工作流（推荐CLI）！

缺点：源码模式
