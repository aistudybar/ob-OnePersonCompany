# 学习过程：【搜】、【学】、【思】、【习】、【教】

> {金句}学习过程就是写文档的过程！
> 

从搜集网络资源、汇总为自己的学习文档（指的是笔记不是文章！否则跟不上大脑思维速度）及它们之间的连接关系，然后遍历复习，到最后的、最好的验证学习成果的方法—-教（如果能把别人教会教懂，才算你真正掌握的该知识点）

1. 【随手记】记录下头脑中一闪而过的念头、灵感等，记在【随手记】中；
2. 【搜】记录到【网络资源索引】
3. 【学】进一步详细学习，生成一个子页面进行知识点整理（如截屏、关键字、标注和短评等等）
4. 【思】在大段空闲学习时间时，扩充完善思考后，整理到Notion页面，并建立相关连接。小火花到大火苗、上下左右联系（父子左右邻居和相关链接，类似根系中的种子和根茎，为遍历提供方便；
5. 【习#补习】补充和修正知识结构和内容；
6. 【习#复习】用遗忘曲线强化记忆；
7. 【教#文章】利用AI生成文章并发布到信息树博客，自己以学生的角度反复阅读、提出疑问，并补充修改；
8. 【教#视频】利用AI生成视频，自己以学生的角度反复阅读、提出疑问，并补充修改文章和视频；
